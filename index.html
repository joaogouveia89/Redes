Histórico

  • Em 1844 - Samuel Morse (Código Morse)
  • Em 1876 - Graham Bell (Telefone Comp. Bell)
  • Até 1960/1970 - Computação de grande porte
  • Processamento centralizado
  • Administração Centralizada
  • Uso de terminais
  • Em 1969 - Projeto Arpanet --> Percursor da internet
  • Início dos anos 70 - Rede de computadores Aloha
  • Meados dos anos 70 - Rede padrão ethernet
    • Empresa Xerox
  • Meados dos anos 70 - Rede padrão token ring
    • Empresa IBM
  • Início dos anos 80 - Personal Computer
  • Nos anos 80, 90 e 2000 surgiram sistemas operacionais para redes de microcompuitadores: Novell Netware, Windows NT, Windows 2000, Windows XP, Windows 2003, Linux, Windows 2008, etc.
  • Anos 90 - Desenvolvimento do comércio eletrônico e sistemas de segurança para redes de computadores
  • Anos 2000 - Utilização em larga escala de redes wireless, convergência dos sistemas de telefonia de voz e dados, explosão do uso das redes sociais por meio de dispositivos móveis de comunicação

Extra: Tipos de cabos

  • Coaxial
  • Par trançado: De fácil instalação e barato, porém pode sofrer facilmente interferência eletromagnética e suas informações podem ser facilmente roubadas
  • Fibra: Seguro e rápido, é a melhor opção entre os 3, porém seu preço é elevado

Componentes básicos de uma rede

  • Computadores servidores(arquivo, impressão, firewall, aplicativos)
  • Computadores clientes
  • Dispositivos de conectividade(Hub, Switch, Router)
  • Sistemas operacionais de rede
  • Softwares, aplicativos e utilitários
  • Cabeamento/Wireless
